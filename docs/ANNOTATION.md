# Annotation workflow

## Recommended path

1. Configure keys in **Settings** (if collecting)
2. Add data via **Search** or **Import**
3. Label in **Annotate**

## Annotate layout

- Post context
- Comment and controls
- Platform ToS guidance

## Keyboard shortcuts

- `0-6`: label
- `1-3`: severity
- `U`: uncertain
- `Enter` / `Space`: save next
- `Backspace`: undo
- Arrow keys: navigate

## Config sources

- Taxonomy: `config/taxonomy.yaml`
- ToS guide text: `config/tos_guide/*.yaml`
