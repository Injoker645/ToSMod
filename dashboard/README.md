# Dashboard

Flask UI for:

- Settings
- Search
- Database
- Annotate
- Import
- Experiments (Advanced)

## Run (from `C:\Uni\ToSMod`)

```powershell
python -m tosmod serve
# or
python -m dashboard
```

Do not run from `C:\Uni\Thesis` (private thesis workspace).

For non-technical users, double-click `Launch-ToSMod.bat` in the repo root and choose a launch mode.
